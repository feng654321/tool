023_how to use wireshark

● 如何用wireshark将很大的报文分解为很小的报文

  1. Enter the wireshark directory in cmd of windows. 'cd c:\Program Files\Wireshark'
  2. editcap.exe -c num -F pcap input_file output_file

● 如何快速筛选51001的报文

  1. 明确消息id

#define SH_MIN_API_FROM_TC           (SH_MIN_API) /* 166 */

#define SH_MAX_NUMBER_API_FROM_TC    (13)

#define SH_REGISTER_USER             (SH_MIN_API_FROM_TC + 0)

#define SH_DEREGISTER_USER              (254)

#define SH_N_UNITDATA_REQUEST        (SH_MIN_API_FROM_TC + 1)

#define SH_N_STATE_REQUEST           (SH_MIN_API_FROM_TC + 2)

#define SH_INCLUDE_CO_SERVICE

#define SH_INCLUDE_CO_CLASS3_SERVICE

#ifdef SH_INCLUDE_CO_SERVICE

#define SH_N_CONNECT_REQUEST         (SH_MIN_API_FROM_TC + 3)

#define SH_N_CONNECT_RESPONSE        (SH_MIN_API_FROM_TC + 4)

#define SH_N_CONNECT_REQUEST_TYPE1   (SH_MIN_API_FROM_TC + 5)

#define SH_N_CONNECT_REQUEST_TYPE2   (SH_MIN_API_FROM_TC + 6)

#define SH_N_DATA_REQUEST            (SH_MIN_API_FROM_TC + 7)

#define SH_N_DISCONNECT_REQUEST      (SH_MIN_API_FROM_TC + 8)

#define SH_N_INFORM_REQUEST          (SH_MIN_API_FROM_TC + 9)

例如 针对 cr  SH_N_CONNECT_REQUEST 对应值为 166 + 3 --> A9 代码中 sag_build_n_connect_request 函数可以查阅到其他字段所在的位置，在所有报文中搜索某特定字段可以使用一下filter 筛选

data contains a9:02:08:70:00:01:00:01:58:f8

corr_id 63576 --> f858  -字节序转换->f858

具体分析可以参考附件 – tcpdump分析示例

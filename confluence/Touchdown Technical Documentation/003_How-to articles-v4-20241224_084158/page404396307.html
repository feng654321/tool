012_How to create AMR call on RNC

● 硬件部署

● IP配置

配置命令：

3.1 FPC 的配置

ip addr add 33.40.10.100/24 dev eth1

ip addr add 33.40.10.101/24 dev eth1

 

3.2 RNC的配置：

add networking ether /EIPU-0 port sfp19 iface uptraffic

add networking ether /EIPU-1 port sfp19 iface uptraffic

add networking ether /EIPU-2 port sfp20 iface uptraffic

add networking ether /EIPU-3 port sfp20 iface uptraffic


 

add networking address /QNUP-0 iface uptraffic ip-address 33.40.10.1/24 

add networking address /QNUP-0 iface uptraffic ip-address 33.40.10.2/24 

 

add networking ipbr ipbr-id 1 ipbr-name ipbr1 committed-bandwidth 1000000 route-bandwidth 1000000 committed-sig-bandwidth 100 committed-dcn-bandwidth 100

add networking ipbr ipbr-id 4 ipbr-name ipbr1 committed-bandwidth 1000000 route-bandwidth 1000000 committed-sig-bandwidth 100 committed-dcn-bandwidth 100

 

add networking ipro ipbr-id 1 iface uptraffic ip-address 33.40.10.1 owner /QNUP-0 

add networking ipro ipbr-id 4 iface uptraffic ip-address 33.40.10.2 owner /QNUP-0

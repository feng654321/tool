032_How to install a 64bit FPC

1.设置本地源（设置之后yum命令才可以用,请查看下面页面中的"Usgae Instruction"部分中的三步）
http://10.56.126.75/.help/centos.html

2. Install svn
    yum install httpd httpd-devel subversion mod_dav_svn mod_auth_mysql

3. Install git
    yum install git

4. Install gcc
    yum -y install gcc

5. Install python
        5.1 Get source code package from official website:
        https://www.python.org/ftp/python/2.7.3/Python-2.7.3.tar.bz2
        5.2 Extract source code from package
        tar -jxvf Python-2.7.3.tar.bz2
        5.3 Configure the makefile
        cd Python-2.7.3
        ./configure
        5.4 Compile and install
        make
        make install
        5.5 Check whether the new installed python take successfully
        python --version


6. Robot framework installation
        6.1 Download Robot framework source code package
        https://pypi.python.org/packages/source/r/robotframework/robotframework-2.8.7.tar.gz
        6.2 Extract source code from the package
        tar xzvf robotframework-2.8.7.tar.gz
         6.3  Install the robot framework module
        cd robotframework-2.8.7
        python setup.py install
        6.4 Check whether the module installed successfully
        pybot --version

7.  pycrypto installation
        7.1 Download package  
        https://pypi.python.org/packages/source/p/pycrypto/pycrypto-2.6.1.tar.gz
       7.2 Extract source code from the package
        tar xzvf pycrypto-2.6.1.tar.gz
       7.3 install
        cd pycrypto-2.6.1
        python setup.py install
       7.4 check the install result
         python
         >>> from Crypto.Cipher import AES


8.  ecdsa installation
        8.1 Download package
        https://pypi.python.org/packages/source/e/ecdsa/ecdsa-0.13.tar.gz
        8.2 Extract source code from the package
        tar xzvf ecdsa-0.13.tar.gz
        8.3 Install
        cd ecdsa-0.13
        python setup.py install
        8.4 check the install result
        python
        >>> import ecdsa


9. Paramiko installation
       9.1 Get the paramiko library from :
        ftp://10.68.157.151/robot/paramiko/paramiko.1.15.zip,
       9.2 Extract source code from the package
        unzip paramiko.1.15.zip
       9.3 install
        cd paramiko
        python setup.py install
       9.4  check the install result
        python
        >>> import paramiko


10. install other library(请注意至上而下的安装顺序，否则会安装出问题)
        scp -r root@10.68.157.17:/root/sourcecode/ ./
        10.1 install every library with below way:
        all need install library:
             setuptools-18.5     
             pytz-2015.7
             Babel-2.1.1         
             curl-7.45.0   //这一个是另外的装法(./configure， make,  make install)
             iso8601-0.1.10      
             monotonic-0.4       
             netaddr-0.7.18      
             pexpect-3.3         
             prettytable-0.7.2   
             PyYAML-3.11         
             pyaml-15.8.2        
             requests-2.8.1      
             scapy-2.3.1         
             six-1.10.0          
             SSHLibrary-2.1.2    
             wrapt-1.10.5        
            10.2 cd every destination directory
            python setup.py install

11. install lksctp
    yum list lksctp*
    yum install lksctp-tools.x86_64
    yum install lksctp-tools-devel.x86_64
    yum install lksctp-tools-doc.x86_64

12. install sctp(FPC重启之后就没有了，这里一定要注意)
    lsmod |grep -i sctp
    modinfo sctp
    cd /lib/modules/2.6.32-431.el6.x86_64/kernel/net/sctp/
    modprobe sctp
    lsmod |grep -i sctp  //再次查询一下

13. JAVA
        13.1 install with binary package
                yum install java-1.7.0-openjdk

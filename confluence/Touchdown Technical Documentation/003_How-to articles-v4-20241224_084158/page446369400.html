036_How to apply for a new PRB and let its running

In Design phase, maybe need to creat a new PRB, How to apply for new PRB and let it running normally?

Step-by-step guide

  1. Apply for new PRB name in UNRA
    1. LINK:  http://unra.inside.nokiasiemensnetworks.com/index.php
    2. If you do not have permission to create name ,you can contact Wang, Jing C. (Nokia - CN/Hangzhou) <jing.c.wang@nokia.com>
    3. login in with your nsn account and apply name:
      1. click namespaces
      2. select files and program blocks in DX200& IPA products
      3. click create new name
      4. check new PRB name whether is available or not(UNRA required that the top three of name cannot be repeated)
        1. PRB name must 6 characters
        2. enter PRB name and look for it, if the top three of name is repeated, we need name a new name for PRB;if it is not occupied , it indicator you can use this name.


      5. Fill paremeter for new PRB and submit
    4. send mail to Support, See (Nokia - Global) <see.support@nokia.com> for  ACCEPTE it.
  2. Apply for family Id
    1. you can send mail to Wang, Jing C. (Nokia - CN/Hangzhou) <jing.c.wang@nokia.com>  for apply family id when new PRB application completed.
    2. the family id name must 8 characters and top six of name is PRB name.
  3. How to add ldif for new RPB
    1. add releevant ldif parameter
      2. template file:
    2. copy to destinnation path:
      1. /$tar_dir$rti_mom_ldifdir/  : /etc/opt/nsn/ldapfiles/ldif_in


  4. How to modify deployment ldif:(the file in /etc/opt/nsn/ldapfiles/ldif_in)
  5. How to deploy and  run 
    1. add ldif parameter for new PRB 
    2. add deployment change 
      1. LINK: http://web.sw3ci.dynamic.nsn-net.net/ILItfChg/interfaceChanges/deploymentchange/add
      2. upload ldif and submit
    3. contact deployment team ，deployment team will modify their ldif and deploy new PRB runing in RU
    4. compile image and install it for testing
    5. merge code with deployment together
  6. Follow CCI and make sure the code in trunk ，install build and regression cases.
    1. image Link: http://10.56.118.71/CloudIL/vrnc/ci/latest/
